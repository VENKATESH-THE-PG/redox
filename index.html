About the Project

The humanoid robot is designed to move based on human gestures, leveraging AI and machine learning. This project aims to enhance human-robot interaction by training the robot to recognize and replicate human movements.

Features

  • Gesture-based control using AI.
  • Real-time machine learning for gesture recognition.
  • Built-in AI assistant for voice commands.
  • ESP32-CAM integration for video streaming.
  • Lightweight and efficient robot hardware design.

Future Use

This project has the potential to be applied in various industries including healthcare, education, and service sectors where robots can assist with tasks by mimicking human actions.
